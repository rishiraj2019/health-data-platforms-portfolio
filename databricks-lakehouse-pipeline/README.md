# Databricks Lakehouse Pipeline

## Overview
A modern data engineering pipeline built using Databricks Community Edition and Delta Lake.

## Tools Used
- Databricks CE
- SQL
- Python (optional)
- Delta Lake

## Data
- Kaggle clinical datasets (public)

## Steps
1. Upload raw CSV  
2. Clean + transform  
3. Write to Delta Lake  
4. Query using SQL  
5. Optional: Create dashboard  

## Skills Demonstrated
- Lakehouse architecture  
- Delta Lake  
- SQL pipelines
