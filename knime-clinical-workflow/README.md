# KNIME Clinical Data Workflow

## Overview
A low‑code ETL workflow that transforms clinical trial data into CDISC‑like or FHIR‑ready structures.

## Tools Used
- KNIME Analytics Platform (free)

## Data
- CDISC sample datasets  
- Kaggle clinical datasets  

## Steps
1. Ingest raw clinical data  
2. Clean + standardize  
3. Map to CDISC‑like structure  
4. Export final dataset  

## Skills Demonstrated
- Low‑code ETL  
- Clinical data modeling  
- Workflow automation
